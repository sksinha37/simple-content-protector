Easily protect your WordPress website content. Disable right-click, keyboard shortcuts, and F12 Developer Tools to prevent unauthorized copying and secure your content.  

== Description ==  
Are you worried about unauthorized copying of your valuable website content? Simple Content Protector is the ultimate solution for securing your WordPress website. This lightweight plugin prevents users from copying text, images, and source code by disabling right-click, text selection, and common keyboard shortcuts like CTRL+C and CTRL+U.  

**Key Features:**  
- Disable right-click to prevent content copying.  
- Block F12 Developer Tools key to hide your source code.  
- Stop common keyboard shortcuts for copy, cut, paste, save, and print.  
- Disable image dragging to protect visual content.  
- Allow administrators to disable protection for their own sessions.  

**Additional Protection Includes:**  
- Disables Mouse Right-Click.  
- Disables `CTRL+C` – Copy.  
- Disables `CTRL+X` – Cut.  
- Disables `CTRL+V` – Paste.  
- Disables `CTRL+U` – View Source.  
- Disables `CTRL+A` – Select All.  
- Disables `CTRL+S` – Save.  
- Disables `CTRL+P` – Print.  
- Disables Image Dragging.  

Simple Content Protector is optimized for performance and works out of the box, ensuring your content remains secure without any complicated setup.  

== Benefits ==  
- Protects your original content from unauthorized use.  
- Reduces the risk of content theft, plagiarism, and misuse.  
- Enhances the security of your WordPress website.  

== Installation ==  
1. Download the `simple-content-protector` plugin.  
2. Upload the plugin folder to the `/wp-content/plugins/` directory.  
3. Activate the plugin through the 'Plugins' menu in WordPress.  

Once activated, your content is automatically secured without any additional configuration.  

== Frequently Asked Questions ==  

= How does this plugin protect my content? =  
Simple Content Protector disables right-click, text selection, and common keyboard shortcuts like CTRL+C, CTRL+U, and F12. It also blocks image dragging, ensuring your content is not easily copied or accessed.  

= Can administrators bypass the protection? =  
Yes, administrators can disable the protection for themselves to ensure smooth website management and editing.  

= Does this plugin slow down my website? =  
No, Simple Content Protector is lightweight and optimized for performance, ensuring no noticeable impact on your website’s speed.  

= Does the plugin offer 100% content protection? =  
While the plugin blocks most common methods of copying, no solution can guarantee 100% protection. However, it provides a strong deterrent against casual content theft.
